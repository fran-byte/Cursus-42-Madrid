```
Nombre de la asignación  : ft_rrange
Archivos esperados   : ft_rrange.c
Funciones permitidas: malloc
--------------------------------------------------------------------------------

Escribe la siguiente función:

int     *ft_rrange(int start, int end);

Debe asignar (con malloc()) un arreglo de enteros, llenarlo con valores consecutivos que comienzan en end y terminan en start (¡Incluyendo start y end!), y luego retornar un puntero al primer valor del arreglo.

Ejemplos:

- Con (1, 3) se devolverá un arreglo que contiene 3, 2 y 1.
- Con (-1, 2) se devolverá un arreglo que contiene 2, 1, 0 y -1.
- Con (0, 0) se devolverá un arreglo que contiene 0.
- Con (0, -3) se devolverá un arreglo que contiene -3, -2, -1 y 0.
```
