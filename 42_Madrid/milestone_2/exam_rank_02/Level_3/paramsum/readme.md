```
Nombre de la asignación  : paramsum
Archivos esperados   : paramsum.c
Funciones permitidas: write
--------------------------------------------------------------------------------

Escribe un programa que muestra el número de argumentos que se le pasen, seguido de un salto de línea.

Si no hay argumentos, solo muestra un 0 seguido de un salto de línea.

Ejemplo:

$>./paramsum 1 2 3 5 7 24
6
$>./paramsum 6 12 24 | cat -e
3$
$>./paramsum | cat -e
0$
```
