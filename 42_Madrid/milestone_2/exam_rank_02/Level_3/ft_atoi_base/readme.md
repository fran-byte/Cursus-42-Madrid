```
Nombre de la asignación  : ft_atoi_base
Archivos esperados   : ft_atoi_base.c
Funciones permitidas: Ninguna
--------------------------------------------------------------------------------

Escribe una función que convierte la cadena de caracteres str (base N <= 16) a un entero (base 10) y lo retorna.

Los caracteres reconocidos en la entrada son: 0123456789abcdef.
Estos deben ser ajustados según la base solicitada. Por ejemplo, la base 4 reconoce "0123" y la base 16 reconoce "0123456789abcdef".

Las letras mayúsculas también deben ser reconocidas: "12fdb3" es lo mismo que "12FDB3".

Los signos negativos ('-') son interpretados solo si son el primer carácter de la cadena.

Tu función debe ser declarada como sigue:

int	ft_atoi_base(const char *str, int str_base);
```
