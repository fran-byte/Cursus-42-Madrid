```
Nombre de la asignación  : reverse_bits
Archivos esperados   : reverse_bits.c
Funciones permitidas: Ninguna
--------------------------------------------------------------------------------

Escribe una función que tome un byte, lo invierta bit por bit (como el ejemplo) y devuelva el resultado.

Tu función debe estar declarada de la siguiente manera:

unsigned char	reverse_bits(unsigned char octet);

Ejemplo:

  1 byte
_____________
 0100  0001
     ||
     \/
 1000  0010
```
