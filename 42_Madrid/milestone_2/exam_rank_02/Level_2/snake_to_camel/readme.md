```
Nombre de la asignación  : snake_to_camel
Archivos esperados   : snake_to_camel.c
Funciones permitidas: malloc, free, realloc, write
--------------------------------------------------------------------------------

Escribe un programa que tome una sola cadena en formato snake_case
y la convierta en una cadena en formato lowerCamelCase.

Una cadena en formato snake_case es una cadena en la que cada palabra está en minúsculas, separada por un guion bajo "_".

Una cadena en formato lowerCamelCase es una cadena donde cada palabra comienza con una letra mayúscula, excepto la primera.

Ejemplos:

$>./snake_to_camel "here_is_a_snake_case_word"
hereIsASnakeCaseWord

$>./snake_to_camel "hello_world" | cat -e
helloWorld$

$>./snake_to_camel | cat -e
$
```
